# Brian_Callahan_p2

## Project Description

Mulesoft American Flights API allows user to have full access creating, updating, and removing flights.
 
## Supported Operations
 * Get all flights/ flight by ID
 * Create a flight
 * Replace flight by ID
 * Delete flight by ID

## Technologies Used
 
 * Anypoint Studio
 * JDK 8
 * Postgresql
 * Anypoint platform
 * Maven
